The Heading component creates `<h1>`, `<h2>`, etc. tags based on the level prop set. Use the size prop to change the size to a different level depending on your needs.

```js live
<div>
    <Heading level={1} size={1}>H1 Level 1 Size 1</Heading>
    <Heading level={2} size={2}>H2 Level 2 Size 2</Heading>
    <Heading level={3} size={3}>H3 Level 3 Size 3</Heading>
    <Heading level={4} size={4}>H4 Level 4 Size 4</Heading>
    <Heading level={5} size={5}>H5 Level 5 Size 5</Heading>
    <Heading level={6} size={6}>H6 Level 6 Size 6</Heading>
</div>
```
